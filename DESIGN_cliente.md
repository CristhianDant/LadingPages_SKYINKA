# Skydive Inka Perú - Design System Specification

## 1. Visión General

Identidad visual deportiva y aventurera con paleta azul-celeste corporativo. Estética moderna y dinámica que transmite confianza, seguridad y emoción. Combinación de tipografía manuscrita (Rock Salt) para impacto emocional y sans-serif (Rubik) para claridad informativa.

## 2. Paleta de Colores

| Nombre Lógico | Valor HEX | Uso Asignado |
|---------------|-----------|--------------|
| Primary Blue | #4D7CEA | Botones principales, acentos, títulos héroe |
| Primary Dark | #0059CB | Hover states, énfasis fuerte |
| Primary Light | #7AD7FF | Elementos decorativos, acentos claros |
| Celeste BG | #F1F4F9 | Fondos de secciones alternas |
| White | #FFFFFF | Fondos principales, textos sobre oscuro |
| Black | #000000 | Textos principales |
| Gray Dark | #333333 | Textos secundarios |
| Gray Medium | #666666 | Textos de soporte |
| Gray Light | #EBEBEB | Bordes, separadores |
| Gray Text | #4B4B4B | Textos cuerpo |
| Pink Accent | #CC3366 | Acentos especiales, hover links |
| WhatsApp Green | #25D366 | Botón WhatsApp |

## 3. Tipografía

| Familia | Pesos | Aplicación |
|---------|-------|------------|
| Rock Salt | Regular 400, Semi-bold 600 | Títulos héroe, headlines emotivos, marca |
| Rubik | Semi-bold 600 | Títulos de sección, cuerpo de texto, navegación |

**Jerarquía:**
- H4 Títulos héroe: 22.4px Rock Salt, color Primary Blue
- H2 Secciones: 32px Rubik 600
- H2 Grande: 48px Rubik 600
- Body: 19.2px Rubik 600
- Navegación: Rubik uppercase

## 4. Espaciado y Efectos

- **Border-radius:** Valores moderados (varía por componente)
- **Box-shadow:** Sombra sutil en tarjetas
- **Transiciones:** Suaves en hover de botones y links
- **Fondos alternos:** Secciones con Celeste BG (#F1F4F9) para ritmo visual
